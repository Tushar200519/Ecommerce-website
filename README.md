# LUXE — Premium E-Commerce Website

A fully functional e-commerce frontend built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies.

## Features

- 20+ products with images and descriptions
- Product search, filter by category, sort by price/rating
- Shopping cart with coupon code support (WELCOME20)
- Checkout with order placement
- Admin panel (login: admin / admin123)
  - Dashboard with revenue stats
  - Order management with status tracking
  - Stock management
  - Add / Edit / Delete products
- Footer with Contact, Track Order, Returns, FAQ
- Browser back/forward navigation support
- localStorage for persistent data

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Unsplash images (requires internet)
- Google Fonts (Cormorant Garamond, DM Sans, DM Mono)

## How to Run

1. Clone the repository
2. Open `index.html` in any browser
3. No server or build step required

## Project Structure

├── index.html   → All pages (SPA)
├── style.css    → Complete stylesheet
└── app.js       → All JavaScript logic
